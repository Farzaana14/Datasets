# Datasets

PRJNA1142891 is a BioProject accession number associated with a study on the molecular epidemiology of mpox (monkeypox) in South Africa in 2024.

The study is titled: Preliminary report on the molecular epidemiology of mpox, South Africa 2024, and can be found at: https://www.biorxiv.org/content/10.1101/2024.11.07.622575v1.full.pdf
